## App Description :scroll:
Diggraffiti is a mobile app that allows anyone to go anywhere in the world and construct virtual artworks using digital paintbrush tools. Once published, these virtual artworks can be viewed by other users within the app and those users can choose to either like the artwork or even create their own!
